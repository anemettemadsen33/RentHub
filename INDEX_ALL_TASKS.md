## üöÄ API Endpoints Summary

### Authentication (`/api/v1`)
- POST `/register` - User registration
- POST `/login` - User login
- POST `/logout` - User logout
- GET `/me` - Current user
- POST `/verify-email` - Verify email
- POST `/verify-phone` - Verify phone
- POST `/2fa/enable` - Enable 2FA
- POST `/forgot-password` - Password reset

### Properties (`/api/v1/properties`)
- GET `/` - List properties
- GET `/{id}` - Property details
- POST `/` - Create property
- PUT `/{id}` - Update property
- DELETE `/{id}` - Delete property
- GET `/featured` - Featured properties
- GET `/search` - Search properties

### Bookings (`/api/v1/bookings`)
- GET `/` - List bookings
- GET `/{id}` - Booking details
- POST `/` - Create booking
- PUT `/{id}/cancel` - Cancel booking
- PUT `/{id}/confirm` - Confirm booking

### Payments (`/api/v1/payments`)
- POST `/process` - Process payment
- GET `/invoices` - List invoices
- GET `/invoices/{id}` - Invoice details
- GET `/invoices/{id}/download` - Download PDF
- POST `/payouts` - Request payout

### Reviews (`/api/v1/reviews`)
- GET `/` - List reviews
- GET `/{id}` - Review details
- POST `/` - Create review
- PUT `/{id}` - Update review
- DELETE `/{id}` - Delete review
- POST `/{id}/helpful` - Mark helpful
- POST `/{id}/respond` - Owner response

### Notifications (`/api/v1/notifications`)
- GET `/` - List notifications
- GET `/unread-count` - Unread count
- PUT `/{id}/read` - Mark as read
- PUT `/mark-all-read` - Mark all read
- DELETE `/{id}` - Delete notification

### Messaging (`/api/v1/conversations`)
- GET `/` - List conversations
- GET `/{id}` - Conversation details
- POST `/` - Start conversation
- POST `/{id}/messages` - Send message
- PUT `/messages/{id}/read` - Mark read

### Wishlist (`/api/v1/wishlists`)
- GET `/` - List wishlists
- POST `/` - Create wishlist
- POST `/{id}/items` - Add property
- DELETE `/{id}/items/{propertyId}` - Remove property
- GET `/shared/{token}` - View shared wishlist

### Calendar (`/api/v1/properties/{id}`)
- GET `/availability` - Get availability
- POST `/block-dates` - Block dates
- DELETE `/unblock-dates` - Unblock dates
- GET `/ical` - Export iCal
- POST `/calendar/sync` - Sync calendars

### Search (`/api/v1`)
- POST `/map/search-radius` - Radius search
- POST `/map/search-bounds` - Bounds search
- POST `/saved-searches` - Save search
- GET `/saved-searches` - List searches
- POST `/saved-searches/{id}/alert` - Set alert

### Verification (`/api/v1`)
- POST `/verification/government-id` - Upload ID
- GET `/verification/status` - Check status
- POST `/properties/{id}/verify` - Verify property

### Analytics (`/api/v1`)
- GET `/owner/dashboard/overview` - Owner overview
- GET `/owner/dashboard/revenue-reports` - Revenue
- GET `/tenant/dashboard/overview` - Tenant overview
- GET `/tenant/dashboard/spending-reports` - Spending

### Multi-language (`/api/v1`)
- GET `/languages` - List languages
- GET `/translations` - Get translations
- GET `/detect-language` - Auto-detect
- POST `/translations` - Create translation (admin)
- POST `/translations/import` - Import translations (admin)

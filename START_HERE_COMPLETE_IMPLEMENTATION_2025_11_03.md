## ğŸ” Troubleshooting

### Common Issues & Solutions

#### 1. "OAuth not working"
```bash
# Clear config cache
php artisan config:clear

php artisan tinker
```

```bash

php artisan passport:keys --force
```

#### 4. "Frontend build errors"
```bash
# Clear and reinstall
rm -rf node_modules package-lock.json
npm install
npm run build
```
